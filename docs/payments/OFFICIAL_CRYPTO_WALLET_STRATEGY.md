# Official Crypto Wallet Strategy

Official crypto wallet payments are accepted only for 8thGuard digital services.

## Supported public rails
- BTC
- ETH / EVM rails
- USDT TRC20
- USDT BEP20
- TON / USDT on TON
- XRP, including destination tag when required
- Solana

## Safety
- Confirm exact network before sending.
- Wrong-network payments may be unrecoverable.
- Never request seed phrases, private keys, or wallet login credentials.
- Publish only public receiving addresses through approved env vars.
