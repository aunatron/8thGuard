# Logo And Visual Identity

Current assets live in `public/assets/`.

Use the logo as a first-viewport brand signal on public pages. The visual system should feel premium, direct, and trust-heavy:
- high contrast
- restrained green/black foundation
- gold for paid/premium emphasis
- clean layouts built for scanning
- no playful fraud-proof claims

Do not make the brand feel local-only. Local payment rails can be mentioned without limiting the global brand.
